# HermesYoutubeCurator
This repo contains some orchestration glue, skills, meta-prompts and guardrails to accomplish : An agent which browses your personal youtube page and compiles a list of videos that might seem interesting to you currently from the home page.

The project constitution lives at `.specify/memory/constitution.md`. The working
default is a simple Python-centric stack close to repositories such as
`earlgreyhot1701D/vigil-crest`: mostly Python API libraries, Playwright where
browser automation is required, and Markdown-based skills/agent guidance.
